# CineWorkflo - AI Prompts for Filmmakers

A curated library of professional AI video prompts for Runway, Pika, and other AI video tools.

## Features

- 150+ tested prompts for commercial use
- Shot-to-Prompt AI tool (upload image, get prompt)
- Smart search and filtering
- One-click copy to clipboard
- Workflow templates

## Tech Stack

- React + Vite
- Tailwind CSS
- Supabase (database)
- Stripe (payments)
- Claude AI (vision analysis)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Deployed to Netlify with automatic builds from main branch.
